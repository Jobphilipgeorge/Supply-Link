SUPPLY-LINK – Supply Chain Management System
📌 Overview
SUPPLY-LINK is a web-based supply chain management system designed to streamline and automate supplier, inventory, and order management processes. The system helps businesses efficiently track stock levels, manage suppliers, and process orders, reducing manual errors and improving operational efficiency.

---

🚀 Features
- 📦 Inventory Management – Track and update stock levels in real-time  
- 🧑‍💼 Supplier Management – Add, update, and manage supplier details  
- 🛒 Order Processing – Create and manage orders efficiently  
- 🔍 Data Tracking – Maintain records using a structured database  
- 💻 User-Friendly Interface – Simple and responsive frontend  

---

🛠️ Tech Stack
- Backend: Java  
- Frontend: HTML, CSS, JavaScript  
- Database: MySQL  
- Tools: Git, VS Code
